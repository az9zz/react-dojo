# 001. Array Flatten (扁平化嵌套数组)

## 题目描述

实现一个函数，将一个嵌套多层的数组 `array` "拉平"（flatten），变成一个一维数组。

## 示例

**示例 1：**

- **输入:** `[1, [2, [3, 4], 5], 6]`
- **输出:** `[1, 2, 3, 4, 5, 6]`

**示例 2：**

- **输入:** `[1, 2, [3, 4, [5]]]`
- **输出:** `[1, 2, 3, 4, 5]`

**示例 3：**

- **输入:** `[1, [2, 3], [4, [5, 6]]]`
- **输出:** `[1, 2, 3, 4, 5, 6]`

---

## 解法一：ES6 `flat()` 方法 (终极解法)

在实际工作中，如果环境允许（现代浏览器和 Node.js 都支持），这永远是最佳选择。简洁、高效、可读性强。

### 核心思想

`Array.prototype.flat(depth)` 方法会按照一个可指定的深度 `depth` 递归地将所有子数组合并到新数组中。

- `depth`: 指定要提取嵌套数组的结构深度，默认为 `1`。
- 我们可以传入 `Infinity` 作为 `depth`，这样无论数组嵌套多少层，它都可以完全扁平化。

### 复杂度分析

- **时间复杂度**: O(n)，其中 `n` 是数组中所有元素的总数。
- **空间复杂度**: O(n)，需要返回一个新数组。

---

## 解法二：递归 (经典思路)

这是最经典的面试思路，考察对递归的理解。

### 核心思想

遍历数组，对每个元素进行判断：

- 如果元素不是数组，直接将其加入结果数组。
- 如果元素是数组，**递归调用**扁平化函数处理这个子数组，然后将返回的结果合并到最终的结果数组中。

### 算法步骤

1. 初始化一个空的结果数组 `result`。
2. 遍历输入数组 `arr`。
3. 对于每个元素 `item`，使用 `Array.isArray(item)` 判断其类型。
4. 如果是数组，使用扩展运算符 `...` 或 `concat` 将递归调用的结果合并到 `result`。
5. 如果不是数组，直接 `push` 到 `result`。
6. 返回 `result`。

---

## 解法三：使用 `reduce` (函数式编程)

这种方法是递归思想的函数式编程体现，代码更简洁，但理解起来可能需要一点思考。

### 核心思想

使用 `Array.prototype.reduce()` 方法来遍历数组并构建最终的扁平化数组。

- `reduce` 的回调函数接收两个参数：`accumulator` (累加器，即我们正在构建的结果数组 `result`) 和 `currentValue` (当前元素 `item`)。
- 在回调函数中，我们同样判断 `item` 是否是数组。
  - 如果是，递归调用扁平化函数，并将其结果与 `accumulator` 合并。
  - 如果不是，直接将 `item` 与 `accumulator` 合并。
- `reduce` 的初始值是一个空数组 `[]`。

## 解法四：迭代 + 栈 (非递归思路)

这种方法使用栈来模拟递归的调用过程，可以避免在嵌套层数非常深时可能出现的“栈溢出”问题。

### 核心思想

1. 创建一个栈 `stack`，并将整个输入数组的**副本**（避免修改原数组）压入栈中。
2. 创建一个结果数组 `result`。
3. 当栈不为空时，循环执行以下操作：
   - 从栈顶弹出一个元素 `item`。
   - 判断 `item` 的类型：
     - 如果**不是**数组，将其`unshift` (头插法) 到结果数组 `result` 中。
     - 如果**是**数组，将其所有元素**从右到左**依次压入栈中。

**为什么要从右到左入栈，然后头插入结果数组？**
因为栈是“后进先出”(LIFO)。我们从栈顶弹出 `[a, b, c]`，为了保持 `a, b, c` 的原始顺序，我们需要先把 `c` 压入栈，再压 `b`，再压 `a`。这样弹出的时候顺序就是 `a, b, c`。同样，为了保持最终结果的顺序，我们使用 `unshift` 将元素添加到结果数组的头部。

_（也可以正序入栈，最后 `result.reverse()`，或者正序入栈+尾插法，最后`reverse()`）_

## 解法五：`toString()` + `split()` (巧妙但不推荐的 "Hack")

这是一种非常取巧的方法，利用了数组的 `toString()` 特性。

### 核心思想

1. 调用数组的 `toString()` 方法，它会自动将多维数组转换为一个用逗号分隔的字符串。例如 `[1, [2, 3]]` 会变成 `"1,2,3"`。
2. 使用 `split(',')` 将这个字符串重新变回一个字符串数组 `['1', '2', '3']`。
3. 遍历这个字符串数组，将每个元素用 `Number()` 或 `parseInt()` 转换回数字。

### 缺点

- **只能处理纯数字数组**。如果数组中包含 `null`, `undefined`, 对象或其他类型，`toString()` 的结果会不符合预期。
- 性能通常不如其他方法。
- 在面试中可以作为一种“脑筋急转弯”提及，但**绝不能作为主要或推荐解法**。
