# 647. Palindromic Substrings (回文子串)

## 题目描述

给你一个字符串 `s` ，请你统计并返回这个字符串中 **回文子串** 的数目。

**回文字符串** 是正着读和倒过来读一样的字符串。
**子串** 是字符串中的由连续字符组成的一个序列。

具有不同开始位置或结束位置的子串，即使是由相同的字符组成，也会被视作不同的子串。

## 示例

**示例 1：**

- **输入:** `s = "abc"`
- **输出:** `3`
- **解释:** 三个回文子串: "a", "b", "c"。

**示例 2：**

- **输入:** `s = "aaa"`
- **输出:** `6`
- **解释:** 6个回文子串: "a", "a", "a", "aa", "aa", "aaa"。

## 约束条件

- `1 <= s.length <= 1000`
- `s` 由小写英文字母组成。

---

## 解法一：动态规划 (Dynamic Programming)

### 核心思想

我们定义一个二维布尔数组 `dp[i][j]`，它表示字符串 `s` 中从索引 `i` 到 `j` 的子串 `s[i...j]` **是否是回文串**。

**状态转移方程**:
要判断 `dp[i][j]` 是否为 `true`，需要满足两个条件：

1. `s[i]`必须等于 `s[j]` (首尾字符相同)。
2. 内部的子串 `s[i+1...j-1]` 也必须是回文串，即 `dp[i+1][j-1]` 为 `true`。

所以，状态转移方程为：`dp[i][j] = (s[i] === s[j]) && dp[i+1][j-1]`。

**基础情况 (Base Case)**:

- **单个字符**: `dp[i][i]` 肯定是 `true` (例如 "a")。
- **两个字符**: `dp[i][i+1]` 为 `true` 当且仅当 `s[i] === s[i+1]` (例如 "aa")。

从状态转移方程可以看出，计算 `dp[i][j]` 依赖于 `dp[i+1][j-1]`，即左下角的值。因此，我们需要改变遍历顺序，从下到上、从左到右地填充 `dp` 表。

**算法步骤**:

1. 初始化 `count = 0`。
2. 创建一个 `n x n` 的二维布尔数组 `dp`。
3. 外层循环 `i` 从 `n-1` 到 `0` (从下到上)。
4. 内层循环 `j` 从 `i` 到 `n-1` (从左到右)。
5. 在循环中，判断 `s[i...j]` 是否是回文：
   - 如果 `s[i] === s[j]`：
     - 如果子串长度小于等于 2 (`j - i <= 1`)，则它一定是回文。
     - 否则，它的回文性取决于 `dp[i+1][j-1]`。
   - 如果 `dp[i][j]` 被确定为 `true`，则 `count++`。
6. 返回 `count`。

### 复杂度分析

- **时间复杂度**: O(n²)，需要填充 `n x n` 的 `dp` 表。
- **空间复杂度**: O(n²)，需要 `dp` 表的空间。

## 解法二：中心扩展法 (Center Expansion)

这是一种更巧妙且空间效率更高的方法。

### 核心思想

回文串都是关于其中心对称的。一个长度为 `n` 的字符串，有 `2n - 1` 个可能的“回文中心”。

- **奇数长度的回文串**: 中心是一个字符，有 `n` 个 (例如 "aba"，中心是 "b")。
- **偶数长度的回文串**: 中心是两个字符之间的空隙，有 `n - 1` 个 (例如 "abba"，中心是 "b" 和 "b" 之间)。

我们可以遍历这 `2n - 1` 个中心，并从每个中心向两边扩展，计算以该中心能构成多少个回文串。

### 算法步骤

1. 初始化 `count = 0`。
2. 遍历 `i` 从 `0` 到 `n-1`，对于每个 `i`：
   a. **以 `s[i]` 为中心** (奇数长度)，调用一个辅助函数 `expandAroundCenter(s, i, i)`。
   b. **以 `s[i]` 和 `s[i+1]` 之间为中心** (偶数长度)，调用辅助函数 `expandAroundCenter(s, i, i+1)`。
3. 将两个辅助函数返回的回文串数量累加到 `count` 中。
4. 返回 `count`。

### 辅助函数 `expandAroundCenter(s, left, right)`

- 该函数接收字符串 `s` 和一个中心（由 `left` 和 `right` 指针表示）。
- `while (left >= 0 && right < s.length && s[left] === s[right])`:
  - 只要指针没有越界且左右字符相等，说明找到了一个回文串，`count++`。
  - 向两边扩展指针：`left--`, `right++`。
- 返回这个中心扩展出的回文串数量。

### 复杂度分析

- **时间复杂度**: O(n²)。有 `2n-1` 个中心，每个中心最多扩展 `n/2` 次。
- **空间复杂度**: O(1)。我们只需要常数级别的额外空间。
