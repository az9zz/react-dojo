# 3. Longest Substring Without Repeating Characters (无重复字符的最长子串)

## 题目描述

给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长子串** 的长度。

## 示例

**示例 1:**

- **输入:** s = "abcabcbb"
- **输出:** 3
- **解释:** 因为无重复字符的最长子串是 "abc"，所以其长度为 3。

**示例 2:**

- **输入:** s = "bbbbb"
- **输出:** 1
- **解释:** 因为无重复字符的最长子串是 "b"，所以其长度为 1。

**示例 3:**

- **输入:** s = "pwwkew"
- **输出:** 3
- **解释:** 因为无重复字符的最长子串是 "wke"，所以其长度为 3。请注意，你的答案必须是 **子串** 的长度，"pwke" 是一个子序列，不是子串。

## 约束条件

- `0 <= s.length <= 5 * 10^4`
- `s` 由英文字母、数字、符号和空格组成。

---

## 解题思路

这个问题是典型的可以使用 **滑动窗口 (Sliding Window)** 思想解决的题目。

我们可以使用两个指针，`left` 和 `right`，来表示一个“窗口”，这个窗口内的子串是当前我们正在考察的无重复字符的子串。

1.  **数据结构选择**：我们需要一个数据结构来快速判断一个字符是否已经存在于当前窗口中。`Set` 是完美的选择，因为它提供了 O(1) 时间复杂度的添加、删除和查找操作。

2.  **算法步骤**：
    - 初始化 `left = 0`, `maxLength = 0`，以及一个空的 `charSet`。
    - 用 `right` 指针从左到右遍历整个字符串 `s`，这个过程可以看作是不断地扩大窗口的右边界。
    - 在每一步，检查 `s[right]` 是否已经存在于 `charSet` 中。
      - **如果存在**，说明窗口内出现了重复字符。我们需要收缩窗口的左边界。不断地从 `charSet` 中移除 `s[left]`，并且将 `left` 指针右移 (`left++`)，直到 `charSet` 中不再包含 `s[right]` 这个重复字符为止。
      - **如果不存在**，说明当前字符是新的。
    - 将 `s[right]` 添加到 `charSet` 中。
    - 更新最大长度：`maxLength = Math.max(maxLength, right - left + 1)`。这个表达式就是当前窗口的长度。
    - 循环结束后，`maxLength` 就是最终答案。

3.  **复杂度分析**:
    - **时间复杂度**: O(n)。虽然代码里有一个 `while` 循环嵌套在 `for` 循环里，但每个字符最多被 `left` 和 `right` 指针访问两次。所以总的时间复杂度是线性的。
    - **空间复杂度**: O(k)，其中 k 是字符串中不同字符的数量。在最坏的情况下（所有字符都不同），空间复杂度是 O(n)。如果字符集有限（例如ASCII），则可以认为是 O(1)。
