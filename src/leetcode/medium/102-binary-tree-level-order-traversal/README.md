# 102. Binary Tree Level Order Traversal (二叉树的层序遍历)

## 题目描述

给你一个二叉树的根节点 `root` ，返回其节点值的 **层序遍历** 。 （即逐层地，从左到右访问所有节点）。

## 示例

**示例 1：**
![Tree Example](https://assets.leetcode.com/uploads/2021/02/19/tree_level.jpg)

- **输入:** `root = [3,9,20,null,null,15,7]`
- **输出:** `[[3],[9,20],[15,7]]`

**示例 2：**

- **输入:** `root = [1]`
- **输出:** `[[1]]`

**示例 3：**

- **输入:** `root = []`
- **输出:** `[]`

## 约束条件

- 树中节点数目在范围 `[0, 2000]` 内
- `-1000 <= Node.val <= 1000`

---

## 解题思路：广度优先搜索 (BFS)

二叉树的层序遍历是广度优先搜索 (BFS) 的最直接应用。BFS 的核心是使用一个**队列 (Queue)** 数据结构来辅助遍历。

### 核心思想

1.  **队列**: 我们需要一个队列来存储将要访问的节点。队列的特性是“先进先出”(First-In, First-Out, FIFO)，这完美地契合了我们逐层处理的需求。
2.  **逐层处理**:
    - 首先，将根节点 `root` 入队。
    - 当队列不为空时，开始循环。在每一轮循环中，我们处理**一整层**的节点。
    - 为了处理一整层，我们首先记录下**当前队列的长度 `levelSize`**。这个 `levelSize` 就是当前层所包含的节点数。
    - 然后，我们进行一个 `for` 循环，次数为 `levelSize`。在这个循环中：
      - 从队列中取出一个节点（出队）。
      - 将该节点的值存入一个表示当前层结果的临时数组 `currentLevel` 中。
      - **关键**: 检查该节点是否有左子节点和右子节点。如果有，就将它们**依次入队**，为下一层的处理做准备。
    - 当 `for` 循环结束后，`currentLevel` 数组就包含了当前层所有节点的值。我们将它加入最终的结果数组中。
3.  **循环继续**: 重复步骤 2，直到队列为空，表示所有层都已处理完毕。

### 算法步骤

1. 如果根节点 `root` 为空，直接返回空数组 `[]`。
2. 初始化一个结果数组 `result = []`。
3. 初始化一个队列 `queue`，并将根节点 `root` 入队。
4. `while (queue.length > 0)` 循环：
   a. 获取当前队列的长度 `levelSize = queue.length`。
   b. 初始化当前层的结果数组 `currentLevel = []`。
   c. 循环 `levelSize` 次 (`for i from 0 to levelSize-1`):
   i. 从队列头部取出一个节点 `node = queue.shift()`。
   ii. 将 `node.val` 加入 `currentLevel`。
   iii. 如果 `node.left` 存在，`queue.push(node.left)`。
   iv. 如果 `node.right` 存在，`queue.push(node.right)`。
   d. 将 `currentLevel` 加入 `result`。
5. 返回 `result`。

## 复杂度分析

- **时间复杂度**: O(N)，其中 N 是树中的节点数。因为每个节点只会被访问（入队和出队）一次。
- **空间复杂度**: O(W)，其中 W 是树的最大宽度。在最坏的情况下（完全二叉树），最后一层可能有 N/2 个节点，所以空间复杂度是 O(N)。
