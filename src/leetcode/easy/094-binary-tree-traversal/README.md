# Binary Tree Traversal (二叉树遍历)

二叉树的深度优先遍历 (DFS) 主要有三种方式：前序、中序和后序。它们的区别在于**根节点的访问时机**。每种遍历都可以通过递归和迭代（使用栈）两种方式实现。

**示例树:**

```
      1
     / \
    2   3
   / \
  4   5
```

- **前序遍历结果**: `[1, 2, 4, 5, 3]`
- **中序遍历结果**: `[4, 2, 5, 1, 3]`
- **后序遍历结果**: `[4, 5, 2, 3, 1]`

---

## 1. 前序遍历 (Pre-order Traversal)

**顺序: 根 -> 左 -> 右**

### 递归实现

这是最直观的实现方式。

1. 访问根节点 `root`。
2. 递归地对左子树 `root.left` 进行前序遍历。
3. 递归地对右子树 `root.right` 进行前序遍历。

### 迭代实现 (使用栈)

1. 初始化一个空栈 `stack` 和一个结果数组 `result`。
2. 将根节点 `root` 压入栈中。
3. 当栈不为空时，循环执行：
   a. 弹出栈顶节点 `node`。
   b. 将 `node.val` 加入 `result`。
   c. **关键**: 先将**右子节点**压入栈中（如果存在）。
   d. 再将**左子节点**压入栈中（如果存在）。
   (因为栈是后进先出，所以要先压右再压左，才能保证弹出时先处理左)

---

## 2. 中序遍历 (In-order Traversal)

**顺序: 左 -> 根 -> 右**
中序遍历二叉搜索树 (BST) 会得到一个有序序列。

### 递归实现

1. 递归地对左子树 `root.left` 进行中序遍历。
2. 访问根节点 `root`。
3. 递归地对右子树 `root.right` 进行中序遍历。

### 迭代实现 (使用栈)

迭代实现相对复杂，需要模拟递归的调用栈。

1. 初始化一个空栈 `stack` 和一个结果数组 `result`，一个指针 `current` 指向 `root`。
2. 当 `current` 不为空或 `stack` 不为空时，循环执行：
   a. **`while (current)`**: 只要 `current` 不为空，就不断将 `current` 压入栈中，并深入其左子树 (`current = current.left`)，直到最左边的节点。
   b. 当 `current` 为空时，说明左子树已探到底。从栈中弹出一个节点 `node`。
   c. 将 `node.val` 加入 `result` (相当于访问“根”)。
   d. 将 `current` 指针指向 `node` 的右子树 (`current = node.right`)，准备处理右子树。

---

## 3. 后序遍历 (Post-order Traversal)

**顺序: 左 -> 右 -> 根**
后序遍历常用于计算表达式树或释放树的节点。

### 递归实现

1. 递归地对左子树 `root.left` 进行后序遍历。
2. 递归地对右子树 `root.right` 进行后序遍历。
3. 访问根节点 `root`。

### 迭代实现 (使用栈)

后序遍历的迭代实现是最巧妙的。我们可以利用前序遍历的思路进行改造。

- **前序遍历**: 根 -> 左 -> 右
- **后序遍历**: 左 -> 右 -> 根

观察发现，后序遍历的顺序 `[左, 右, 根]`，反过来就是 `[根, 右, 左]`。这个顺序与前序遍历 `[根, 左, 右]` 非常相似！

**算法步骤**:

1. 按照“根 -> 右 -> 左”的顺序遍历树，并将结果存入一个临时数组。这个过程和前序遍历的迭代实现几乎一样，只是压栈的顺序是**先左后右**。
2. 将这个临时数组**反转**，就得到了“左 -> 右 -> 根”的后序遍历结果。
