# 20. Valid Parentheses (有效的括号)

## 题目描述

给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

有效字符串需满足：

1.  左括号必须用相同类型的右括号闭合。
2.  左括号必须以正确的顺序闭合。

## 示例

**示例 1：**

- **输入:** `s = "()"`
- **输出:** `true`

**示例 2：**

- **输入:** `s = "()[]{}"`
- **输出:** `true`

**示例 3：**

- **输入:** `s = "(]"`
- **输出:** `false`

**示例 4：**

- **输入:** `s = "([)]"`
- **输出:** `false`

**示例 5：**

- **输入:** `s = "{[]}"`
- **输出:** `true`

## 约束条件

- `1 <= s.length <= 10^4`
- `s` 仅由括号 `'()[]{}'` 组成。

---

## 解题思路：栈 (Stack)

这个问题具有“后进先出” (Last-In, First-Out, LIFO) 的特性，这正是栈数据结构的典型应用场景。

### 核心思想

我们可以遍历字符串，利用一个栈来存储遇到的左括号。

- 当遇到一个**左括号** (`(`, `[`, `{`) 时，我们将其**压入栈**中。
- 当遇到一个**右括号** (`)`, `]`, `}`) 时，我们需要检查栈顶的元素：
  - 如果此时栈是空的，说明没有对应的左括号，直接返回 `false`。
  - 如果栈不为空，我们**弹出栈顶**的左括号，并检查它是否与当前遇到的右括号是相匹配的一对。
    - 如果不匹配，返回 `false`。
    - 如果匹配，则继续遍历。

### 算法步骤

1.  初始化一个空栈 `stack`。
2.  创建一个 `Map` 来存储括号的配对关系，例如 `map = { ')': '(', ']': '[', '}': '{' }`。这样可以方便地查找右括号对应的左括号。
3.  遍历输入字符串 `s` 的每一个字符 `char`。
4.  **如果 `char` 是一个左括号** (即 `map` 的值中包含它)：
    - 将其压入 `stack` 中。
5.  **如果 `char` 是一个右括号** (即 `map` 的键中包含它)：
    - 检查 `stack` 是否为空。如果为空，或者 `stack.pop()` 弹出的元素不等于 `map.get(char)`，则说明不匹配，返回 `false`。
6.  遍历结束后，**检查 `stack` 是否为空**。
    - 如果 `stack` 为空，说明所有左括号都被正确地闭合了，返回 `true`。
    - 如果 `stack` 不为空，说明有多余的左括号没有被闭合（例如 `"("` 或 `"{["`），返回 `false`。

## 复杂度分析

- **时间复杂度**: O(n)，其中 n 是字符串的长度。我们只需要遍历一次字符串。
- **空间复杂度**: O(n)，在最坏的情况下（例如，字符串全是左括号 `((((...`），我们需要将所有字符都压入栈中。
